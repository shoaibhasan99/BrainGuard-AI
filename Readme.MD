# BrainGuard-AI 🧠

BrainGuard-AI is an AI-based system designed to detect neurological disorders such as Alzheimer's Disease and Multiple Sclerosis using deep learning models.

## 🚀 Features
- Alzheimer detection using Swin Transformer
- MS detection using ResNet50
- Medical image analysis
- User-friendly interface

## 🛠️ Tech Stack
- Python
- TensorFlow / PyTorch
- OpenCV
- Deep Learning Models
